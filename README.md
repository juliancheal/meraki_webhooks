# Meraki Webhooks

https://developer.cisco.com/meraki/webhooks/

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'meraki_webhooks'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install meraki_webhooks
